在 github pages 上使用 custom domain

Posted on 2016-12-17 |

github pages 在之前有提到是一個很方便讓工程師來寫部落格的好方法，但是它的 domain 被限制在 username.github.io，如果我們想要使用自己的 domain 呢？這當然是可以的！github 就有一篇 指引 在說明這如何完成這件工作．而我自己之前一直想要買個 domain 加在 blog 上面，但最近都滿（ㄓㄨㄤ）忙，所以就一直拖．但今天終於來把它搞定啦．這中間找了許多方法一一介紹流程在底下．
